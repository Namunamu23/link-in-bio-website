# /assets/fonts

Self-hosted display font. `gochi-hand.woff2` is included as a placeholder — Gochi Hand from Google Fonts (SIL Open Font License). Replace it with a licensed hand-lettered font you own, then update the `src:` in `styles/main.css` if the filename changes.

Do not hotlink Google Fonts — self-hosting keeps the site private-by-default and avoids third-party requests.
