# /assets/illustrations

Drop real hand-drawn SVGs here — clouds, drops, custom button shapes.

To swap the placeholder shapes in `index.html`:
1. Save your SVG here (for example `cloud-tl.svg`).
2. Replace the corresponding inline `<svg>` in `index.html` with an `<img>` referencing the file, or paste the new path into the existing `<svg>`.
